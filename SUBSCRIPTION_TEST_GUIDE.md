http://localhost:3000/test/subscription

## 🎯 테스트 시나리오

### 1. Free 플랜 시나리오
- **`free-at-limit`** 🔴 - 광고 계정 한도 도달 (1/1)
  - 광고 계정 연결 시도 → 업그레이드 모달 표시
  - 팀원 초대 시도 → 업그레이드 모달 표시

- **`free-empty`** 🟢 - 여유 있음 (0/1)
  - 광고 계정 1개 연결 가능
  - 2개째 연결 시도 시 차단

### 2. Standard 플랜 시나리오
- **`standard-warning`** 🟡 - 경고 구간 (4/5, 무료체험 3일 남음)
  - 80% 도달 시 경고 표시
  - 1개 더 연결 가능
  - 무료 체험 종료 임박 알림

- **`standard-at-limit`** 🔴 - 한도 도달 (5/5)
  - 광고 계정/팀원 추가 불가
  - 업그레이드 모달 표시

- **`standard-normal`** 🟢 - 정상 (2/5)
  - 3개 더 연결 가능

### 3. Pro 플랜 시나리오
- **`pro-unlimited`** 💎 - 무제한 (15개 연결)
  - 광고 계정 무제한 연결
  - 진행바에 "무제한" 표시

### 4. 무료 체험 시나리오
- **`trial-ending-1day`** ⏰ - 체험 종료 임박 (1일 남음)
  - 긴급 알림 표시
  - 결제 정보 등록 유도

- **`trial-fresh`** 🎉 - 체험 시작 (14일 남음)
  - 체험 안내 표시

## 🎬 테스트 방법

### 1. 시나리오 선택
드롭다운에서 원하는 시나리오 선택

### 2. 액션 테스트
- **"광고 계정 연결 시도"** 버튼 클릭
  - ✅ 성공: Toast 알림 + 카운트 증가
  - ❌ 실패: Toast 에러 + 업그레이드 모달

- **"팀원 초대 시도"** 버튼 클릭
  - ✅ 성공: Toast 알림 + 카운트 증가
  - ❌ 실패: Toast 에러 + 업그레이드 모달

### 3. UI 확인 사항
- [ ] 진행바 색상 (80% 이상 주황색, 100% 빨간색)
- [ ] 경고 메시지 표시
- [ ] 업그레이드 모달 디자인
- [ ] Toast 알림 위치 및 내용
- [ ] 무제한 플랜 표시

### 4. 초기화
"현재 시나리오 초기화" 버튼으로 원래 상태로 복원
